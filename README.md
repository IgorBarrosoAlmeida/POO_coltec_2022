<h1>Programação Orientada a objetos - 2022</h1>
<img width="40" height="50" src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/java/java-original.svg" />

Este repositório guarda parte dos códigos feitos durante o ano de 2022 na disciplina AEDS (Algoritmos e estrutura de dados), do curso de Desenvolvimento de Sistemas do Colégio técnico da UFMG (Coltec).

A disciplina AEDS foi ministrada pela professora Virginia Mota e tem como objetivo primario aprender sobre as principais estruturas de dados tanto teoricamente quanto na prática e após isso introduzir os alunos à programação orientada a objetos. Esse repositório guarda os códigos referentes ao aprendido sobre POO. As aulas dadas estão listadas a seguir:

1 e 2. Introdução a OO
3. Sintax basica de java
4. POO usando java
5. Arrays
6. Modificadores
7. Herança
8. Classes Abstratas
9. Interfaces

Os arquivos estão organizados de forma que, cada uma das aulas tem sua respectiva pasta, e em cada uma delas temos os arquivo *.java e um Exercicios.txt descrevendo os exercícios realizados no arquivo *.java.

Nessa parte da disciplina foram dados dois trabalhos sobre o assunto, ambos estão em repositórios exclusivos para eles.
